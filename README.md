# Price Monitor Bot

Учебный backend-проект на Python для отслеживания цен товаров через внешний API
с сохранением истории в базе данных и уведомлениями в Telegram.

Проект находится в активной разработке и используется мной как практический кейс
для изучения backend-разработки, работы с API, SQL и сервисной архитектуры.

## Что делает сервис

Сервис предназначен для мониторинга цен товаров по их идентификатору.

На текущий момент реализовано:
- получение данных о товаре из внешнего API;
- сохранение карточки товара и истории цен в SQL-базу данных;
- хранение «сырых» ответов API для отладки и анализа;
- расчет изменения цены относительно предыдущего значения;
- Telegram-бот с командой для ручного обновления цены и просмотра динамики;
- механизм подписок и автоматических уведомлений при изменении цены.

## Используемые технологии

- Python
- SQLite
- SQL
- Работа с внешними API (HTTP)
- Telegram Bot API
- Хранение конфигурации через ".env"
- Планировщик фоновых задач

Я учусь backend-разработке и хотел собрать проект, который будет похож

Проект развивается постепенно, по мере изучения мною новых тем.
