# Price Monitor Bot

Учебный backend-проект на Python для отслеживания цен товаров через внешний API
с сохранением истории в базе данных и уведомлениями в Telegram.

Проект находится в активной разработке и используется мной как практический кейс
для изучения backend-разработки, работы с API, SQL и сервисной архитектуры.

## Что делает сервис

Сервис предназначен для мониторинга цен товаров по их идентификатору.

На текущий момент реализовано:
- получение данных о товаре из внешнего API;
- сохранение карточки товара и истории цен в SQL-базу данных;
- хранение «сырых» ответов API для отладки и анализа;
- расчет изменения цены относительно предыдущего значения;
- Telegram-бот с командой для ручного обновления цены и просмотра динамики;
- механизм подписок и автоматических уведомлений при изменении цены.

## Используемые технологии

- Python
- SQLite
- Работа с внешними API (HTTP)
- Telegram Bot API
- Хранение конфигурации через ".env"
- Планировщик фоновых задач

## Запуск проекта локально

1. Клонирование репозитория

Склонируйте репозиторий и перейдите в папку проекта.

```bash
git clone https://github.com/Arseniiiiiiiiy/price-monitor-bot.git
cd price-monitor-bot
```

2. Создание виртуального окружения

Виртуальное окружение используется для изоляции зависимостей проекта от глобально установленных библиотек.

```bash
python -m venv .venv
```

3. Активация виртуального окружения (Windows)

После активации окружения все команды python и pip будут выполняться внутри проекта.

```bash
.\.venv\Scripts\Activate.ps1
```

Если окружение активировано корректно, в начале строки терминала появится пометка (.venv).

4. Установка зависимостей

Все необходимые библиотеки перечислены в файле requirements.txt.

```bash
pip install -r requirements.txt
```

5. Создание файла конфигурации .env

В корне проекта необходимо создать файл .env и указать в нём настройки приложения.
Пример содержимого файла:

```env
BOT_TOKEN=токен_telegram_бота
DB_PATH=data.sqlite3
```

6. Запуск приложения

После выполнения всех предыдущих шагов приложение можно запустить командой:

```bash
python main.py
```
